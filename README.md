Add SPSS icon launcher on Ubuntu

* **Step 1**: Install SPSS

* **Step 2**: Run this command on a terminal

```
sudo curl -o- https://raw.githubusercontent.com/JAlbertoGonzalez/spss-linux-launcher/main/spss.sh | bash
```
